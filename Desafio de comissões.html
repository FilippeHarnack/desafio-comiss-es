2) Lançamento de movimentações de estoque

Requisitos:

  • movimentar produtos do JSON (entrada/saída)

  • cada movimentação tem:

    • id único

    • descrição

  • ao final, retornar estoque final do produto

Programa (Python)

python
import uuid

class ControleEstoque:
    def __init__(self, dados_estoque: dict):
        # indexa por codigoProduto
        self.produtos = {
            p["codigoProduto"]: {
                "descricaoProduto": p["descricaoProduto"],
                "estoque": int(p["estoque"])
            }
            for p in dados_estoque["estoque"]
        }
        self.movimentacoes = []  # histórico

    def movimentar(self, codigo_produto: int, quantidade: int, descricao: str):
        if codigo_produto not in self.produtos:
            raise ValueError("Produto não encontrado.")

        if quantidade == 0:
            raise ValueError("Quantidade não pode ser zero.")

        atual = self.produtos[codigo_produto]["estoque"]
        novo = atual + quantidade  # entrada=positivo, saída=negativo

        if novo < 0:
            raise ValueError(
                f"Estoque insuficiente. Atual={atual}, tentativa de saída={-quantidade}"
            )

        mov_id = str(uuid.uuid4())
        self.produtos[codigo_produto]["estoque"] = novo

        self.movimentacoes.append({
            "id": mov_id,
            "codigoProduto": codigo_produto,
            "descricaoMovimentacao": descricao,
            "quantidade": quantidade,
            "estoqueFinal": novo
        })

        return mov_id, novo

3) Juros por atraso (multa 2,5% ao dia)

Interpretação padrão do enunciado: multa simples por dia de atraso sobre o valor original.

  • Se hoje <= vencimento → juros = 0

  • Se atrasou d dias → juros = valor * 0.025 * d

Programa (Python)

python
from datetime import datetime, date

def calcular_juros_simples(valor: float, data_vencimento: str):
    """
    data_vencimento no formato 'YYYY-MM-DD'
    multa de 2,5% ao dia (simples)
    """
    venc = datetime.strptime(data_vencimento, "%Y-%m-%d").date()
    hoje = date.today()

    dias_atraso = (hoje - venc).days
    if dias_atraso <= 0:
        return 0.0, valor, dias_atraso

    juros = valor * 0.025 * dias_atraso
    total = valor + juros
    return round(juros, 2), round(total, 2), dias_atraso

if __name__ == "__main__":
    valor = 1000.00
    vencimento = "2025-11-01"

    juros, total, dias = calcular_juros_simples(valor, vencimento)
    print(f"Dias em atraso: {dias}")
    print(f"Juros: R$ {juros}")
    print(f"Total atualizado: R$ {total}")

Se sua intenção for juros compostos (multa acumulando sobre multa), troque a linha do juros por:
juros = valor * ((1 + 0.025) ** dias_atraso - 1).

2) Lançar movimentações de estoque

Requisitos:

  • entrada = quantidade positiva

  • saída = quantidade negativa

  • gerar id único e salvar descrição

  • impedir estoque negativo

  • retornar estoque final

js
const { randomUUID } = require("crypto");

class ControleEstoque {
  constructor(dadosEstoque) {
    this.produtos = new Map();
    this.movimentacoes = [];

    for (const p of dadosEstoque.estoque) {
      this.produtos.set(p.codigoProduto, {
        descricaoProduto: p.descricaoProduto,
        estoque: Number(p.estoque)
      });
    }
  }

  movimentar(codigoProduto, quantidade, descricaoMovimentacao) {
    if (!this.produtos.has(codigoProduto)) {
      throw new Error("Produto não encontrado.");
    }
    if (quantidade === 0) {
      throw new Error("Quantidade não pode ser zero.");
    }

    const prod = this.produtos.get(codigoProduto);
    const novoEstoque = prod.estoque + quantidade;

    if (novoEstoque < 0) {
      throw new Error(
        `Estoque insuficiente. Atual=${prod.estoque}, saída=${-quantidade}`
      );
    }

    const id = randomUUID();
    prod.estoque = novoEstoque;

    this.movimentacoes.push({
      id,
      codigoProduto,
      descricaoMovimentacao,
      quantidade,
      estoqueFinal: novoEstoque
    });

    return { id, estoqueFinal: novoEstoque };
  }

  consultarEstoque(codigoProduto) {
    if (!this.produtos.has(codigoProduto)) return null;
    return this.produtos.get(codigoProduto).estoque;
  }

  historico() {
    return this.movimentacoes;
  }
}

// Exemplo:
const dadosEstoque = {
  estoque: [
    { codigoProduto: 101, descricaoProduto: "Caneta Azul", estoque: 150 },
    { codigoProduto: 102, descricaoProduto: "Caderno Universitário", estoque: 75 },
    { codigoProduto: 103, descricaoProduto: "Borracha Branca", estoque: 200 },
    { codigoProduto: 104, descricaoProduto: "Lápis Preto HB", estoque: 320 },
    { codigoProduto: 105, descricaoProduto: "Marcador de Texto Amarelo", estoque: 90 }
  ]
};

const ce = new ControleEstoque(dadosEstoque);

console.log(ce.movimentar(101, 30, "Entrada via compra fornecedor"));
console.log(ce.movimentar(104, -50, "Saída para pedido cliente"));

console.log("Estoque final Caneta 101:", ce.consultarEstoque(101));
console.log("Histórico:", ce.historico());

3) Calcular juros por atraso (2,5% ao dia)

Interpretação mais comum: juros simples por dia de atraso sobre o valor original.

js
function calcularJurosSimples(valor, dataVencimentoStr) {
  // dataVencimentoStr no formato "YYYY-MM-DD"
  const vencimento = new Date(dataVencimentoStr + "T00:00:00");
  const hoje = new Date();
  hoje.setHours(0, 0, 0, 0);

  const msPorDia = 24 * 60 * 60 * 1000;
  const diasAtraso = Math.floor((hoje - vencimento) / msPorDia);

  if (diasAtraso <= 0) {
    return { diasAtraso: 0, juros: 0, totalAtualizado: Number(valor.toFixed(2)) };
  }

  const juros = valor * 0.025 * diasAtraso;
  const totalAtualizado = valor + juros;

  return {
    diasAtraso,
    juros: Number(juros.toFixed(2)),
    totalAtualizado: Number(totalAtualizado.toFixed(2))
  };
}

// Exemplo:
const valor = 1000;
const vencimento = "2025-11-01";

console.log(calcularJurosSimples(valor, vencimento));

Se quiser juros compostos, use:

js
const juros = valor * (Math.pow(1.025, diasAtraso) - 1);
